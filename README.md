## konndate コマンド

![test](https://github.com/bannbuu/robosys2025/actions/workflows/test.yml/badge.svg)

`konndate` は、料理のジャンル・調理法・食材を指定すると、対応する料理名を標準出力に 1 行ずつ表示するコマンドです。
エラー時のメッセージは標準エラー出力に出します。

### 使い方

#### コマンドの書式

`./konndate <ジャンル1> <ジャンル2> <ジャンル3>`

- `<ジャンル1>`: `肉` / `魚` / `麺`
- `<ジャンル2>`: `焼く`, `煮る`, `揚げる`, `丼`, `汁あり`, `汁なし` など
- `<ジャンル3>`: `牛`, `豚`, `鳥`, `白身`, `赤身`, `青魚`, `中華`, `和風`, `パスタ`, `アジア` など

#### 使用例

```bash
./konndate 肉 焼く 牛
./konndate 魚 丼 赤身
./konndate 麺 汁なし 中華
'''bash

### 必要なソフトウェア

- Python
  - テスト済みバージョン: 3.8, 3.9, 3.10, 3.11

### テスト環境

- Ubuntu 24.04 LTS（GitHub Actions の `ubuntu-latest` 上で確認）

### データファイル

`konndate` はカレントディレクトリにある次の3つのファイルを読み込みます。

- `foods-肉.txt`
- `foods-魚.txt`
- `foods-麺.txt`

各行は次の形式で記述します。

```text
<食材>,<調理法>:料理名
牛,焼く:ステーキ
牛,揚げる:コロッケ
鳥,丼:親子丼
中華,汁あり:ラーメン
'''bash
