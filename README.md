## food コマンド

コマンドライン引数で料理のジャンルを指定すると，対応する料理名を表示するコマンドです。  
3 つの引数で「大ジャンル」「調理法」「食材」を指定します。

- 第 1 引数: 大ジャンル（例: `肉`）
- 第 2 引数: 調理法（例: `焼く`, `煮る`, `揚げる`, `炒める`, `丼` など）
- 第 3 引数: 食材（例: `牛`, `豚`, `鳥`）

対応する料理が複数ある場合は，1 行ずつすべて表示します。

### 使い方の例

```sh
# 初回のみ実行権限を付与
chmod +x food

# 例: 「肉 / 焼く / 牛」に対応する料理を表示
./food 肉 焼く 牛
# => ステーキ
#    焼肉

# ヘルプの表示
./food --help
データファイル

food コマンドは，同じディレクトリにある次のファイルを参照します。

foods-肉.txt

foods-魚.txt

foods-麺.txt

各ファイルの 1 行は，次の形式になっています。

<食材>,<調理法>:<料理名>


例（foods-肉.txt の一部）:

牛,焼く:ステーキ
牛,焼く:焼肉
豚,焼く:生姜焼き
豚,揚げる:とんかつ
鳥,丼:親子丼


food コマンドは，第 1 引数から適切なファイルを選び，
第 2 引数（調理法）と第 3 引数（食材）に一致する行の料理名を表示します。
